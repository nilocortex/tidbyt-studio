⬛ Tidbyt Studio — Help

Canvas: 64×32 pixel grid matching the Tidbyt LED matrix.

Tools: Pencil, eraser, fill, color picker, line, rect, filled rect, circle.

Shortcuts:

  • B — Pencil
  • E — Eraser
  • G — Fill
  • I — Color Picker
  • L — Line
  • R — Rectangle
  • Ctrl+Z — Undo
  • Ctrl+Y — Redo
  • Ctrl+S — Export PNG

Tabs:

  • Preview — Live preview, widgets, export
  • Frames — Animation frame management
  • Apps — Browse community apps, view schemas
  • Code — Starlark code editor
  • Device — Device dashboard, brightness, installed apps
  • Message — Quick text message push with pixel font
  • Push — Push pixel art to device
