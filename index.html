⬛ Tidbyt Studio — Help

Canvas: 64×32 pixel grid matching the Tidbyt LED matrix.

Tools: Pencil, eraser, fill, color picker, line, rect, filled rect, circle.

Shortcuts:

  • B — Pencil
  • E — Eraser
  • G — Fill
  • I — Color Picker
  • L — Line
  • R — Rectangle
  • Ctrl+Z — Undo
  • Ctrl+Y — Redo
  • Ctrl+S — Export PNG

Frames: Create animation frames, set delay, export as animated WebP/GIF.

Push: Enter your Device ID + API Token from the Tidbyt app to push directly.

Starlark: Generate Pixlet-compatible code to use with pixlet render.

API: POST to api.tidbyt.com/v0/devices/{id}/push with base64 WebP.
